# Simple RDBMS - PesaPal Challenge

A lightweight relational db managment system built from scratch in Python.

## FEatures 
- [] Basic Storage engine
- [] SQL parser (CREATE, INSERT, SELECT, UPDATE, DELETE)
- [] B-tree indexing
- [] REPL interface
- [] Demo web application

## Setup
```bash
python src/repl.py
```
